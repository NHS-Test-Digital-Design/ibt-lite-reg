Register a coronavirus (COVID-19) test

Use this service to register a (COVID-19) test so you can get the test result.

Use a different service if you need to report the result of a free rapid lateral flow test you did at home or work.

You need to register each COVID-19 test to get the results.

If you're eligible for COVID-19 treatments and have symptoms

If you have a free rapid lateral flow test:

  • take it now instead of the PCR test, as you'll get the result quicker
  • report the rapid lateral flow test result on GOV.UK

If you only have a PCR test, continue to register and return the test.

Before you start

You will need:

  • a mobile phone number we can send the result to
  • the test kit barcode - if you're registering a PCR test
  • the Royal Mail barcode - if you're registering a PCR swab test that you will return by post

Get help to register a test

If you cannot use the online service, call 119 (free from mobiles and landlines). Lines are open Monday to Friday 8am to 6pm, Saturday and Sunday 9am to 1pm. 119 provides support in 200 languages.

SignVideo is a free online British Sign Language interpreter service for 119.

You can use the free Be My Eyes app to get help from trained NHS Test and Trace staff. Download the app, go to 'Specialized Help' and select NHS Test & Trace in the 'Personal Health' category.
